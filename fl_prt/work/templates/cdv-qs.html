CatDV QuickSearch

The client needed a tool to enable them to quickly navigate through their media asset database, consisting of over 20,000+ media files.
The application had to clearly display only a set number specific information that they required and be able to return the information as a spreadsheet.
Features of the app include the ability to search using different variables, sorting results and saving the results as an excel spreadsheet or for printable media.
